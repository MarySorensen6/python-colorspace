Examples of Default Palettes
=============================

The `colorspace` package contains a set of default palettes
which can be accessed via palette name.


.. plot::
    :include-source: True
    :width: 100%

    from colorspace import hcl_palettes
    hcl_palettes()
    hcl_palettes(plot = True)
